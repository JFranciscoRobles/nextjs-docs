### Respuestas

#### Usario nuevo creado:

```
{
  code:201,
  msg:{
        Usuario:"Usuario Creado Exitosamente",
        Aviso:"Por favor ingrese el codigo que se la enviado",
        Update:"Actualice sus datos"
      }
}
```
#### Usuario ya registrado
```
{
  code:201,
  msg:"Inicio de sesion exitoso",
  token:token
}
```
#### Éxito:

```
{

  "code": 200,

  "msg": "Usuario creado con exito."

}
```

#### Errores De Validación:

```
{

  "code": 500,

  "msg": "Validation error: El nombre solo puede contener letras y no puede tener espacios en blanco."

}
```

```
{

  "code": 500,

  "msg": "Ese nombre ya esta en uso."

}
```

```
}
```
