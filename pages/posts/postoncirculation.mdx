### **Ruta: (post) (TOKEN) **

https://damp-beyond-72658.herokuapp.com/circulatiopost/:id


agregar por body el un valor booleano, true continua en circulacion, false ya no continua

### Parametros de ejemplo:

```
https://damp-beyond-72658.herokuapp.com/circulatiopost/1

```


```
"judge": true
```

### Respuestas

#### Éxito:

```
{
    "code": 200:
    "msg": "1"
}
```

#### Errores De Validación:

```
{
    "code": 404,
    "msg": "You are not authorized"
}
```

```
{
    "code": 400,
    "msg": "The report does not exist"
}
```


### Notas:
Por Body se debe aregar true o false, en donde true es que si va a continuar en circulacion y false se bloquea en definitiva.
