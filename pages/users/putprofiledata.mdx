### **Ruta: (PUT) (TOKEN)**

https://damp-beyond-72658.herokuapp.com/usuario/update

### Parametros de ejemplo:

```
{
    "name": "Sailedqx",
    "birth": "1997-12-12",
    "email": "sailedgi@gmail.com",
    "phone": "6645684580",
    "address": "Mexico"
}
```
